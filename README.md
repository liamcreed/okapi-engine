# okapi-engine

It should be able to run on windows and on mac.

Features:

- Math libary
    - vectors
    - matrix 4x4
    - quaternions

- Simple 3D rendering engine
  
- GLTF parsing using cgltf and exporting to custom binary format
  
- Simple 3D Animation system
